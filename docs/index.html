        The Labyrinth
        

          
A prefix cache that survives everything

          
Daedalus built the Labyrinth once; only he kept the map. Your agent's giant system prompt
          is prefilled once, ever вҖ” persisted to disk as KV state, matched exactly on every later
          turn, alive across crashes and reboots. Correct even for hybrid-attention models
          (Qwen3.5, Gemma 4) whose caches cannot be trimmed вҖ” where stock servers silently miss.

        

      

      

        The Thread
        

          
Resumable prefill

          
Ariadne's thread marks the path already walked. Mid-prefill checkpoints every 4k tokens
          mean a client timeout, a crash, or OpenCode's hard 300-second deadline never sends you back
          to the entrance вҖ” the retry picks up the thread where it dropped.

        

      

      

        The Middle Path
        

          
A thermal governor, the first of its kind

          
Reads macOS's five-level thermal-pressure signal (no sudo) вҖ” including the
          two levels Apple's public API hides вҖ” and duty-cycles prefill against it. Escalates
          instantly, cools with hysteresis, because a fanless chassis forgives slowly. Three tempers:
          --profile performance | balanced | cool.

        

      

      

        The Herald
        

          
Keepalives that carry news, tools that don't hang

          
SSE heartbeats flow from the first second of a long prefill, carrying progress, thermal
          state, and an ETA. Streamed tool calls are strictly well-formed OpenAI deltas вҖ” indexed,
          never empty вҖ” because one malformed array hangs an agent forever. Any
          OpenAI-compatible client just works: pi, OpenCode, Hermes.
