// inventory

Four items equipped

Built on mlx-lm's public primitives. No monkey-patching, no forks to chase.

ITEM 01 · THE LABYRINTH

Persistent prefix cache

The agent's giant system prompt is prefilled once, ever — saved to disk as KV state, recalled exactly on every later turn, across crashes and reboots. Correct even for hybrid-attention models (Qwen3.5, Gemma 4) where stock servers miss every time.

ITEM 02 · THE THREAD

Resumable prefill

Checkpoints every 4k tokens. A timeout, a crash, or a client's hard deadline never restarts the maze — the retry picks up the thread exactly where it dropped.

ITEM 03 · THE GOVERNOR

First thermal governor in local inference

Reads macOS's five-level thermal-pressure signal live, no sudo — including the two levels Apple's public API hides — and duty-cycles prefill against it. performance · balanced · cool.

ITEM 04 · THE HERALD

Keepalives + clean tool calls

SSE heartbeats with progress, thermal state, and ETA from second one. Reasoning is split into reasoning_content; tool calls stream as strict OpenAI deltas — indexed, never empty. pi, OpenCode, anything OpenAI-compatible just works.
