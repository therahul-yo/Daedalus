01 · the labyrinth

Persistent prefix cache

The agent's giant system prompt is prefilled once, ever — saved to disk as KV state and recalled exactly on every later turn, across crashes and reboots. Correct even for hybrid-attention models (Qwen3.5, Gemma 4) where stock servers silently miss.

02 · the thread

Resumable prefill

Checkpoints every 4k tokens. A timeout, a crash, or a client's hard 300-second deadline never restarts the maze — the retry picks up the thread exactly where it dropped.

03 · the governor

First thermal governor in local inference

Reads macOS's five-level thermal-pressure signal live, no sudo — including the two levels Apple's public API hides — and duty-cycles prefill against it. Three tempers: performance · balanced · cool.

04 · the herald

Keepalives + clean tool calls

SSE heartbeats carry progress, thermal state, and ETA from second one. Reasoning is split into reasoning_content so it never leaks; tool calls stream as strict OpenAI deltas — indexed, never empty. pi, OpenCode, anything compatible just works.
